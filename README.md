# A-FIT

A-FIT is a fitness application focused on helping users track meals and macronutrients from a mobile-friendly experience.

## Overview

This repository currently contains the initial project documentation.

## Goals

- Track daily meals
- Monitor macronutrients (protein, carbs, and fats)
- Keep nutrition logging simple and accessible
